Phần 2 · Phân tích yêu cầu

Yêu Cầu Phi Chức Năng

⚡ Hiệu năng
  • •Tổng thời gian chụp → mở khóa ≤ 3 giây
  • •Xử lý nhận diện trên server ≤ 2 giây/ảnh
  • •Truyền ảnh Wi-Fi ≤ 500ms
🎯 Độ chính xác
  • •True Positive Rate ≥ 90% (ánh sáng bình thường)
  • •False Acceptance Rate ≤ 5%
🔄 Độ ổn định
  • •Hoạt động liên tục ≥ 8 giờ
  • •Tự kết nối lại Wi-Fi khi bị ngắt
  • •Hiển thị lỗi rõ ràng thay vì treo hệ thống
🔒 Bảo mật
  • •Fail-Secure: khóa đóng khi mất điện/mất kết nối
  • •Ghi log mọi trường hợp nhận diện thất bại

Pipeline Nhận Diện Khuôn Mặt

⚖️Hybrid MatchCosine + L2
Combined Score = 0.7 × Cosine Similarity + 0.3 × max(0, 1 − L2_distance / 2.0)

✅ Nếu Combined Score ≥ 0.80 → Xác nhận danh tính. Hybrid metric tăng chính xác so với dùng một metric đơn lẻ.

Phần 4 · Hoạt động

Nguyên Lý Hoạt Động

✅ Nhận diện thành công

  • 1️⃣ Nhấn nút → Camera chụp JPEG
  • 2️⃣ HTTP POST → Server xử lý AI
  • 3️⃣ JSON phản hồi → Tên người dùng
  • 4️⃣ GPIO 3 HIGH → Relay đóng → Khóa mở 3s
  • 5️⃣ LCD: "CHAO MUNG! / [Tên]"
  • 6️⃣ Buzzer: 1 beep ngắn 200ms
  • 7️⃣ Ghi log SUCCESS vào CSV

❌ Từ chối truy cập

  • 1️⃣ Khuôn mặt không khớp CSDL
  • 2️⃣ Relay OFF → Cửa vẫn KHÓA
  • 3️⃣ LCD: "ACCESS DENY"
  • 4️⃣ Buzzer: 2 beep dài cảnh báo
  • 5️⃣ deny_count++
  • 6️⃣ Nếu vượt ngưỡng → Telegram + Còi báo động 30s
  • 7️⃣ Ghi log FAILED vào CSV

Hybrid Metric

Pythondef recognize(self, embedding): for uid, mean in self._means.items(): # Cosine similarity sim = np.dot(embedding, mean) # L2 distance l2 = np.linalg.norm(embedding - mean) # Kết hợp 70% + 30% combined = 0.7*sim + 0.3*max(0,1-l2/2) if best >= 0.80: return uid, best

Cơ Chế An Ninh Thông Minh
